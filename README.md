# Sentinel-2 S2DR3 Inference

This repository demonstrates the use of the **S2DR3 model** to perform **inference on Sentinel-2 satellite imagery** for geospatial and environmental analysis.

## Features
- Processing Sentinel-2 data for Earth Observation (EO)
- Satellite-based environmental and air quality inference
- Reproducible workflow using Python
- Visualization of model outputs
